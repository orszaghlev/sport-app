### How to run the application
Open the `src/main/resources/` folder and run the MySQL scripts on your local MySQL server.

After that, run the Spring Boot application by entering the following command in your terminal:

```
mvn spring-boot:run
```

The final step is to open the `src/react-frontend` folder and enter the following commands:

```
npm install
npm start
```
